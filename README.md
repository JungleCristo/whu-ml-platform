## 基本功能
- 用户管理
    - 用户注册
    - 用户登录
    - 用户密码修改
- 数据管理
    - 数据集上传功能
        - 支持CSV、Excel文件等格式的上传
    - 数据预处理工具
        - 缺失值处理
        - 数据清洗
        - 特征缩放（标准化、归一化）
- 机器学习模型
    - 模型种类
        - 线性回归（Linear Regression）
        - 决策树（Decision Tree）
        - 支持向量机（SVM）
        - K近邻（K-Nearest Neighbors, KNN）
        - 随机森林（Random Forest）
        - 神经网络（Neural Network，基础的前馈神经网络）
    - 模型参数调整：支持调整模型的超参数，如决策树的深度、SVM的核函数等
    - 模型训练
        - 选择数据集和模型进行训练
        - 设置训练参数（学习率、批量大小、训练轮数等）
        - 实时监控训练进度（训练损失、验证损失等）
    - 模型性能评估：提供准确率、精确率、召回率、F1-score等评估指标
